# Project WFP - VitaGuard

Yang harus dilakukan pertama:
1. Clone Repo
2. Buka project di VSCode
3. Jalankan perintah `composer install`
4. Jalankan perintah `copy .env.example .env` 
5. Jalankan perintah `php artisan key:generate`
6. Jalankan perintah `php artisan migrate`

Setiap akan membuat perubahan: 
1) Jalankan perintah `git fetch`, jika ada perubahan jalankan perintah `git pull`
2) Pastikan sudah berada di branch master kemudian, jalankan perintah `git git checkout -b nama-branch`
3) Setelah melakukan perubahan commit perubahan dan jalankan perintah `git push origin nama-branch`
4) Kemudian jalankan perintah `git checkout main`
5) Terakhir jalankan perintah `git branch -d nama-branch`
